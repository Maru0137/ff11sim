# 防御力 (Defense) 計算仕様

参考: [wiki.ffo.jp/html/1685.html](https://wiki.ffo.jp/html/1685.html)

## 総合計算式

```
DEF合計 = int(VIT × 1.5) + Lv + α + 装備DEF + defense_bonus
```

| 項 | 内容 |
|---|---|
| `VIT` | キャラクターの最終 VIT 値（種族 + メインジョブ + サポートジョブ + マスターレベル + メリット + 装備）|
| `Lv` | メインジョブのレベル（1-99）|
| `α` | レベル補正値（下表参照）|
| `装備DEF` | 装備品の防 (DEF) 合計 |
| `defense_bonus` | ジョブ特性・ギフト・JP カテゴリによる防御力ボーナス（下記参照）|

### α (レベル補正)

| レベル範囲 | α |
|---|---|
| Lv1-50 | 8 |
| Lv51-59 | 8 + (Lv - 50) |
| Lv60-90 | 18 |
| Lv91-99 | 18 + int((Lv - 89) / 2) |

具体値:

| Lv | α |
|---|---|
| 50 | 8 |
| 55 | 13 |
| 59 | 17 |
| 60-90 | 18 |
| 91 | 19 |
| 95 | 21 |
| 99 | 23 |

### defense_bonus

```
defense_bonus = ジョブ特性 DefenseBonus(main/sup 最大)
              + ギフト physical_defense
              + JPカテゴリ physical_defense
```

#### ジョブ特性 DefenseBonus

メインジョブとサポートジョブで取得済みのうち**高い方**を採用（加算ではない）。
ランクごとの累積値:

| Rank | 1 | 2 | 3 | 4 | 5 | 6 |
|---|---|---|---|---|---|---|
| 値 | 10 | 22 | 35 | 48 | 60 | 72 |

取得ジョブ・レベル:

- **PLD**: Lv10, 30, 50, 70, 76, 91（最大 rank 6 = +72）
- **WAR**: Lv10, 45, 86（最大 rank 3 = +35）

#### ギフト physical_defense

メインジョブの累計 JP（`total_jp_spent`）に応じてティアごとに加算される。
所属スロットの 4 ティア全解放で累積値が確定。

War の場合（参考実装値）: `[10, 15, 20, 25]` → 全解放で **+70**

ティア解放の閾値（全ジョブ共通、スロット番号により異なる）:

| スロット index | 解放閾値 (累計 JP) |
|---|---|
| 0 | 5, 180, 605, 1280 |
| 1 | 10, 210, 660, 1360 |
| 2 | 20, 245, 720, 1445 |
| 3 | 30, 280, 780, 1530 |
| 4 | 45, 320, 845, 1620 |
| 5 | 60, 360, 910, 1710 |

各ジョブの physical_defense スロット位置と値は `rust/src/job_points.rs` の `JOB_GIFTS` を参照。

#### JP カテゴリ physical_defense

特定カテゴリのランクが直接 physical_defense に寄与するジョブのみ加算。
War のカテゴリは physical_defense に寄与しないので 0。

## 計算例: War99 ML50, VIT=200, 装備DEF=500, JP全マスター

| 項 | 値 |
|---|---|
| `int(VIT × 1.5)` | int(200 × 1.5) = 300 |
| `Lv` | 99 |
| `α` | 23 (Lv91-99 区分) |
| `装備DEF` | 500 |
| `defense_bonus` | 35 (War rank3 Lv86) + 70 (ギフト 2100JP 全解放) + 0 (JPカテゴリ) = 105 |
| **合計** | **300 + 99 + 23 + 500 + 105 = 1027** |

## 実装

| 場所 | 役割 |
|---|---|
| `rust/src/status.rs:calc_defense` | 装備分まで含む基本式 |
| `rust/src/wasm.rs:chara_to_status_result` | `defense_bonus` を加算して `def_total` を算出 |

## 未対応項目

以下は現在未実装または計算外:

- DEF+X% 系装備 (パーセント補正)
- ナイトのカウンタースタンス（ジョブアビリティ補正）
- 食事・薬品・ジョブアビリティ・スピリットリンク等の一時補正
- マスターレベルによる α 補正（実装上 ML は VIT への +1/lv 加算のみ）
